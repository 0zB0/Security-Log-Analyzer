# TraceHawk v0.7.1

TraceHawk v0.7.1 hardens the public distribution and local run defaults.

## Security Corrections

- local Docker ports bind to `127.0.0.1` because local mode has no external authentication;
- self-host instructions select the required production Compose profile;
- synthetic evidence uses RFC 5737 TEST-NET addressing;
- the clean-room exporter rejects the former routable synthetic address;
- Semgrep findings are published to GitHub code scanning through SARIF.

## Public Proof

- regenerated 32-second UI tour and screenshots;
- regenerated HTML and PDF sample reports with checksum manifests;
- complete test, rule-contract, benchmark, dependency, secret, static-analysis, and Docker gates;
- allowlisted public export receipt tied to the verified GitLab source commit.

TraceHawk remains a bounded single-replica portfolio system, not a multi-tenant SIEM.
