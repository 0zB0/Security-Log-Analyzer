# TraceHawk v0.7.0

TraceHawk v0.7.0 is the first proof-complete public engineering release: bounded ingestion, robust
mixed-log parsing, N-step correlation, measured detection quality, performance budgets, RBAC,
observability, recovery tooling, security gates, and visual proof.

## Verified Evidence

- 145 public backend tests with warnings treated as errors;
- 89% line coverage baseline;
- 31 end-to-end detection scenarios;
- 65 of 65 rules with positive contract coverage;
- zero unexpected or missing contract findings;
- Gitleaks and Semgrep public CI gates;
- Python and npm dependency audits;
- reproducible Docker build;
- allowlisted public export with private-marker regression coverage.

## Scope Boundary

TraceHawk remains a single-replica, local-first investigation assistant. It is not a multi-tenant
SIEM, an autonomous response platform, or proof of compromise. Review `docs/limitations.md` and
`docs/threat-model.md` before reuse.
